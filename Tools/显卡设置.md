
#### 方案一：envycontrol (更现代、更简单)

`envycontrol` 是一个轻量级的命令行工具，通过修改内核启动参数来切换模式，非常稳定。


```
yay -S envycontrol
```
    
2. **用法**：
    
    - 查询当前模式：
        ```
        sudo envycontrol --query
        ```
        
    - **切换到集成显卡模式（最省电）**:
        ```
        sudo envycontrol -s integrated
        ```
        
    - 切换到独立显卡模式（性能最强）：
        ```
        sudo envycontrol -s discrete
        ```
        
    - 切换到混合模式（按需使用独显）：
        ```
        sudo envycontrol -s hybrid
        ```
        
    
    切换后需要**重启电脑**才能生效。
    

#### 方案二：optimus-manager (功能更丰富)

`optimus-manager` 是一个后台服务，功能更强大，可以动态切换而无需每次都修改启动参数。

1. **安装 (来自 AUR)**：
    ```
    yay -S optimus-manager
    ```
    
2. **启用服务**：
    ```
    sudo systemctl enable optimus-manager
    ```
    
3. **用法**：
    
    - **切换到集成显卡模式（最省电）**:
        ```
        optimus-manager --switch integrated
        ```
        
    - 切换到独立显卡模式：
        ```
        optimus-manager --switch nvidia
        ```
        
    - 切换到混合模式：
        ```
        optimus-manager --switch hybrid
        ```
        
    
    切换后需要**注销并重新登录**才能生效。
