## [平衡二叉树](https://leetcode.cn/problems/balanced-binary-tree/)

一般的拍脑袋思路是，遍历二叉树，然后对每一个节点计算左右的最大高度。

但是计算一棵二叉树的最大深度也需要递归遍历这棵树的所有节点，如果对每个节点都算一遍最大深度，时间复杂度是比较高的。

所以最好的解法是反过来思考，只计算一次最大深度，计算的过程中在后序遍历位置顺便判断二叉树是否平衡：

对于每个节点，先算出来左右子树的最大高度，然后在后序遍历的位置根据左右子树的最大高度判断平衡性。

```cpp
class Solution {
public:
    bool isBalanced(TreeNode* root) {
        checkBalanced(root);
        return balanced;
    }

private:
    // 记录二叉树是否平衡
    bool balanced = true;

    // 输入一个节点，返回以该节点为根的二叉树的最大深度
    int checkBalanced(TreeNode* root) {
        if (root == nullptr) {
            return 0;
        }
        
        // if (!balanced) {
        // 随便返回一个值即可，旨在结束递归
        //     return -666;
        // }

        int leftMaxDepth = checkBalanced(root->left);
        int rightMaxDepth = checkBalanced(root->right);

        // 后序遍历位置
        // 如果左右最大深度大于 1，就不是平衡二叉树
        if (abs(rightMaxDepth - leftMaxDepth) > 1) {
            balanced = false;
        }

        return 1 + max(leftMaxDepth, rightMaxDepth);
    }
};
```

## [二叉树的所有路径](https://leetcode.cn/problems/binary-tree-paths/)

```cpp
class Solution {
    vector<string> res;
    vector<int> path; // 建议存 int，最后转 string 效率更高

    void traverse(TreeNode* root) {
        if (!root) return;

        // 1. 进入节点：做选择（入栈）
        path.push_back(root->val);

        // 2. 判定：是否是叶子
        if (!root->left && !root->right) {
            res.push_back(formatPath(path));
            // 注意：这里不需要 return，因为后面还有统一的出栈逻辑
        } else {
            // 3. 递归：向深处走
            traverse(root->left);
            traverse(root->right);
        }

        // 4. 离开节点：撤销选择（出栈）
        path.pop_back();
    }

    string formatPath(const vector<int>& p) {
        if (p.empty()) return "";
        stringstream ss;
        for (int i = 0; i < p.size(); ++i) {
            ss << p[i] << (i == p.size() - 1 ? "" : "->");
        }
        return ss.str();
    }

public:
    vector<string> binaryTreePaths(TreeNode* root) {
        res.clear();
        path.clear();
        traverse(root);
        return res;
    }
};
```

“做选择 -> 递归 -> 撤销选择”的模式是解决 **所有回溯类题目**的模板：

```cpp
void traverse(TreeNode* root) {
    if (!root) return;

    // 1. 【进】把当前节点放入路径
    path.push_back(root->val);

    // 2. 【查】如果是叶子，处理结果
    if (!root->left && !root->right) {
        res.push_back(format(path));
    } else {
        // 3. 【走】向深处探索
        traverse(root->left);
        traverse(root->right);
    }

    // 4. 【出】回溯！撤销刚才的选择（重要！）
    path.pop_back();
}
```

## [左叶子之和](https://leetcode.cn/problems/sum-of-left-leaves/)

```cpp
class Solution {
public:
    int sumOfLeftLeaves(TreeNode* root) {
        traverse(root);
        return sum;
    }

    // 记录左叶子之和
    int sum = 0;
    // 二叉树遍历函数
    void traverse(TreeNode* root) {
        if (root == nullptr) {
            return;
        }

        if (root->left != nullptr &&
            root->left->left == nullptr && root->left->right == nullptr) {
            // 找到左侧的叶子节点，记录累加值
            sum += root->left->val;
        }

        // 递归框架
        traverse(root->left);
        traverse(root->right);
    }
};
```

还有一种“统一逻辑”的写法，核心在于：**不让父节点去偷看子节点，而是让子节点带着“身份标签”进场。**

在之前的写法中，父节点必须判断 `root->left` 是否存在，再判断它是不是叶子。这其实打破了递归的独立性。

### 1. 统一递归版：带状态下行

我们给递归函数增加一个 `bool isLeft` 参数。这样，每个节点只需要关心自己：

1. 我是一个叶子吗？
2. 我是从左边来的吗？

```cpp
class Solution {
    int sum = 0;

    void traverse(TreeNode* root, bool isLeft) {
        if (!root) return;

        // 节点只检查自己：我是叶子且我是左孩子吗？
        if (!root->left && !root->right && isLeft) {
            sum += root->val;
            return; // 已经是叶子了，没必要往下递归
        }

        // 往下走时，明确告诉左孩子：你是左边来的；告诉右孩子：你是右边来的
        traverse(root->left, true);
        traverse(root->right, false);
    }

public:
    int sumOfLeftLeaves(TreeNode* root) {
        sum = 0;
        // 根节点虽然没有父节点，但它肯定不是“左叶子”，所以传 false
        traverse(root, false);
        return sum;
    }
};
```

### 2. 这种写法好在哪里？

- **逻辑高度解耦**：父节点不再需要关心子节点的内部结构（是否有左右孩子），它只负责传达“身份”。
- **代码极简**：你会发现 `if` 语句变得非常短，不再有一长串的 `root->left->left` 这种容易看花眼的指针操作。
- **易于扩展**：如果题目改成“求所有右叶子的和”，你只需要把 `isLeft` 判断改一下，逻辑完全通用。

### 3. 归纳：什么时候该用这种“带状态”的 DFS？

当你发现你的递归逻辑里出现了类似 `root->left->left` 这种“**跨级访问**”时，通常就是一个信号：**你应该尝试把信息作为参数传下去。**

常见场景包括：

- **257. 二叉树的所有路径**：传的是 `string path`（路径状态）。
- **98. 验证二叉搜索树**：传的是 `long min, long max`（数值边界状态）。
- **112. 路径总和**：传的是 `targetSum - root->val`（剩余数值状态）。

## [完全二叉树的节点个数](https://leetcode.cn/problems/count-complete-tree-nodes/)

使用框架：注意前序位置是进入节点的时候，所以这里我们乘以了 10, 而后序位置是离开这个节点，向上走（回溯），所以是离开节点，我们要除以 10。

![[Pasted image 20260210220002.png]]

```cpp
class Solution {
    int res = 0;      // 记录最终所有路径的总和
    int track = 0;    // 记录从根节点到当前节点的路径数字（状态）

public:
    int sumNumbers(TreeNode* root) {
        traverse(root);
        return res;
    }

    void traverse(TreeNode* root) {
        if (root == nullptr) {
            return;
        }

        // --- 前序位置 ---
        // 1. 进入节点：更新当前路径代表的数字
        track = track * 10 + root->val;

        // 2. 如果是叶子节点，把当前的路径数字累加到结果中
        if (root->left == nullptr && root->right == nullptr) {
            res += track;
        }

        traverse(root->left);
        // --- 中序位置 ---
        traverse(root->right);

        // --- 后序位置 ---
        // 3. 离开节点：回溯，撤销当前节点对数字的影响
        // 逻辑：123 变回 12，就是 (123 - 3) / 10
        track = (track - root->val) / 10;
    }
};
```

更优雅的写法：不需要全局变量，直接通过递归返回值汇总结果

```cpp
class Solution {
public:
    int sumNumbers(TreeNode* root) {
        return dfs(root, 0);
    }

    // prevSum 是父节点传下来的“累计数字”
    int dfs(TreeNode* root, int prevSum) {
        if (!root) return 0;

        // 1. 计算当前路径代表的数字
        int currentSum = prevSum * 10 + root->val;

        // 2. 如果是叶子节点，这就是一条完整路径的终点，返回这个数字
        if (!root->left && !root->right) {
            return currentSum;
        }

        // 3. 如果不是叶子，递归左右子树，并将结果相加
        return dfs(root->left, currentSum) + dfs(root->right, currentSum);
    }
};
```

## [找树左下角的值](https://leetcode.cn/problems/find-bottom-left-tree-value/)

递归法处理的时候注意显示回溯。

```cpp
class Solution {
    // 记录二叉树的最大深度
    int maxDepth = 0;
    // 记录 traverse 递归遍历到的深度
    int depth = 0;
    TreeNode* res = nullptr;

public:
    int findBottomLeftValue(TreeNode* root) {
        traverse(root);
        return res->val;
    }

private:
    void traverse(TreeNode* root) {
        if (root == nullptr) {
            return;
        }
        // 前序遍历位置
        depth++;
        if (depth > maxDepth) {
            // 到最大深度时第一次遇到的节点就是左下角的节点
            maxDepth = depth;
            res = root;
        }
        traverse(root->left);
        traverse(root->right);
        // 后序遍历位置
        depth--;
    }
};
```

也可以用层序遍历，更加简单，但需要注意右子树先入队，然后左子树入队。这样出队的时候，最后一个出队的一定是最底层最左侧的叶子节点。

```cpp
class Solution {
public:
    int findBottomLeftValue(TreeNode* root) {
        queue<TreeNode*> q;
        q.push(root);
        TreeNode* curr = nullptr;

        while (!q.empty()) {
            curr = q.front();
            q.pop();

            // 关键：先放右子树，再放左子树！
            if (curr->right) q.push(curr->right);
            if (curr->left) q.push(curr->left);
        }

        // 队列中最后一个弹出的节点，一定是最后一行最左边的节点
        return curr->val;
    }
};
```

## [路径总和 II](https://leetcode.cn/problems/path-sum-ii/)

```cpp
class Solution {
  vector<vector<int>> res; // 记录最终所有合法的路径
  vector<int> path;        // 记录当前遍历的路径状态（草稿纸）
  int currentSum = 0;      // 记录当前路径的和（状态）

public:
  vector<vector<int>> pathSum(TreeNode *root, int targetSum) {
    traverse(root, targetSum);
    return res;
  }

  void traverse(TreeNode *root, int targetSum) {
    if (root == nullptr) {
      return;
    }

    // --- 前序位置：做选择 ---
    path.push_back(root->val);
    currentSum += root->val;

    // 判断是否是叶子节点
    if (root->left == nullptr && root->right == nullptr) {
      // 如果路径和等于目标值，拷贝一份当前路径到结果集
      if (currentSum == targetSum) {
        res.push_back(path);
      }
    }

    traverse(root->left, targetSum);
    traverse(root->right, targetSum);

    // --- 后序位置：撤销选择（回溯） ---
    // 离开节点前，必须把状态恢复到进入该节点之前的样子
    currentSum -= root->val;
    path.pop_back();
  }
};
```

## [最大二叉树](https://leetcode.cn/problems/maximum-binary-tree/)

```cpp
class Solution {
  TreeNode *build(vector<int> nums, int lo, int hi) {
    if (lo > hi)
      return nullptr;
    int index = -1, maxVal = INT_MIN;
    for (int i = lo; i <= hi; i++) {
      if (maxVal < nums[i]) {
        maxVal = nums[i];
        index = i;
      }
    }
    TreeNode *root = new TreeNode(maxVal);
    root->left = build(nums, lo, index - 1);
    root->right = build(nums, index + 1, hi);
    return root;
  }

public:
  TreeNode *constructMaximumBinaryTree(vector<int> &nums) {
    return build(nums, 0, nums.size() - 1);
  }
};
```

## [二叉搜索树中的众数](https://leetcode.cn/problems/find-mode-in-binary-search-tree/)

对于 BST, 只要遍历一遍即可，利用中序遍历的特性。

```cpp
class Solution {
    vector<int> res;
    TreeNode* prev = nullptr;
    int count = 0;
    int maxCount = 0;

public:
    vector<int> findMode(TreeNode* root) {
        traverse(root);
        return res;
    }

    void traverse(TreeNode* root) {
        if (!root) return;

        // 1. 左
        traverse(root->left);

        // 2. 中（处理逻辑）
        // 更新当前数字的频率
        if (prev == nullptr || root->val != prev->val) {
            count = 1; // 新数字出现
        } else {
            count++;   // 还是那个数字
        }

        // 更新众数结果
        if (count > maxCount) {
            maxCount = count;
            res.clear();        // 发现更高频率，之前的全作废
            res.push_back(root->val);
        } else if (count == maxCount) {
            res.push_back(root->val); // 频率并列，加入结果
        }

        prev = root; // 更新前驱指针

        // 3. 右
        traverse(root->right);
    }
};
```

如果是普通的二叉树, 什么遍历顺序都无所谓：

```cpp
class Solution {
private:

void searchBST(TreeNode* cur, unordered_map<int, int>& map) { // 前序遍历
    if (cur == NULL) return ;
    map[cur->val]++; // 统计元素频率
    searchBST(cur->left, map);
    searchBST(cur->right, map);
    return ;
}
bool static cmp (const pair<int, int>& a, const pair<int, int>& b) {
    return a.second > b.second;
}
public:
    vector<int> findMode(TreeNode* root) {
        unordered_map<int, int> map; // key:元素，value:出现频率
        vector<int> result;
        if (root == NULL) return result;
        searchBST(root, map);
        vector<pair<int, int>> vec(map.begin(), map.end());
        sort(vec.begin(), vec.end(), cmp); // 给频率排个序
        result.push_back(vec[0].first);
        for (int i = 1; i < vec.size(); i++) {
            // 取最高的放到result数组中
            if (vec[i].second == vec[0].second) result.push_back(vec[i].first);
            else break;
        }
        return result;
    }
};
```

## [二叉树的最近公共祖先](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/)

```cpp
class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
        // Base case: 越过叶子节点，或者直接撞见了 p 或 q
        if (root == nullptr || root == p || root == q) {
            return root;
        }

        // --- 后序位置：先去左右子树里打探消息 ---
        TreeNode* left = lowestCommonAncestor(root->left, p, q);
        TreeNode* right = lowestCommonAncestor(root->right, p, q);

        // --- 核心逻辑：汇总结果 ---
        
        // 情况 1：左边找到了一个，右边也找到了一个
        // 说明 p 和 q 分居 root 的两侧，那么 root 就是最近公共祖先！
        if (left != nullptr && right != nullptr) {
            return root;
        }

        // 情况 2：左右都落空了
        if (left == nullptr && right == nullptr) {
            return nullptr;
        }

        // 情况 3：一边有一边没有
        // 比如左边找到了，右边是空的，那返回左边找到的那个（可能是 p/q，也可能是 LCA）
        return left != nullptr ? left : right;
    }
};
```

## [二叉搜索树的最近公共祖先](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-search-tree/)

```cpp
class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
        // 保证 val1 较小，val2 较大
        int val1 = min(p->val, q->val);
        int val2 = max(p->val, q->val);
        return find(root, val1, val2);
    }

    // 在 BST 中寻找 val1 和 val2 的最近公共祖先节点
    TreeNode* find(TreeNode* root, int val1, int val2) {
        if (root == nullptr) {
            return nullptr;
        }
        if (root->val > val2) {
            // 当前节点太大，去左子树找
            return find(root->left, val1, val2);
        }
        if (root->val < val1) {
            // 当前节点太小，去右子树找
            return find(root->right, val1, val2);
        }
        // val1 <= root->val <= val2
        // 则当前节点就是最近公共祖先
        return root;
    }
};
```

## [二叉搜索树中的插入操作](https://leetcode.cn/problems/insert-into-a-binary-search-tree/)

```cpp
// 定义：在以 root 为根的 BST 中插入 val 节点，返回插入后的根节点
class Solution {
public:
    TreeNode* insertIntoBST(TreeNode* root, int val) {
        if (root == nullptr) {
            // 找到空位置插入新节点
            return new TreeNode(val);
        }

        // 去右子树找插入位置
        if (root->val < val) {
            root->right = insertIntoBST(root->right, val);
        }
        // 去左子树找插入位置
        if (root->val > val) {
            root->left = insertIntoBST(root->left, val);
        }
        // 返回 root，上层递归会接收返回值作为子节点
        return root;
    }
};
```

不要被题目吓到了，认为有多种插入方式，需要各种旋转之类的、、完全而不需要，只要找到叶子节点然后插入即可。

## [删除二叉搜索树中的节点](https://leetcode.cn/problems/delete-node-in-a-bst/)

```cpp
class Solution {
public:
    // 定义：在以 root 为根的 BST 中删除值为 key 的节点，返回完成删除后的根节点
    TreeNode* deleteNode(TreeNode* root, int key) {
        if (root == nullptr) return nullptr;
        if (root->val == key) {
            // 这两个 if 把情况 1 和 2 都正确处理了
            if (root->left == nullptr) return root->right;
            if (root->right == nullptr) return root->left;
            // 处理情况 3
            // 获得右子树最小的节点
            TreeNode* minNode = getMin(root->right);
            // 删除右子树最小的节点
            root->right = deleteNode(root->right, minNode->val);
            // 用右子树最小的节点替换 root 节点
            minNode->left = root->left;
            minNode->right = root->right;
            root = minNode;
        } else if (root->val > key) {
            root->left = deleteNode(root->left, key);
        } else if (root->val < key) {
            root->right = deleteNode(root->right, key);
        }
        return root;
    }

    TreeNode* getMin(TreeNode* node) {
        // BST 最左边的就是最小的
        while (node->left != nullptr) node = node->left;
        return node;
    }
};
```

## [修剪二叉搜索树](https://leetcode.cn/problems/trim-a-binary-search-tree/)

```cpp
class Solution {
public:
    TreeNode* trimBST(TreeNode* root, int low, int high) {
        if (root == nullptr) return nullptr;

        // 情况 1：当前节点太小，弃左保右
        if (root->val < low) {
            // 递归修剪右子树，并直接返回右子树的结果给上一层
            return trimBST(root->right, low, high);
        }

        // 情况 2：当前节点太大，弃右保左
        if (root->val > high) {
            // 递归修剪左子树，并直接返回左子树的结果给上一层
            return trimBST(root->left, low, high);
        }

        // 情况 3：当前节点在范围内，修剪两边并接上
        root->left = trimBST(root->left, low, high);
        root->right = trimBST(root->right, low, high);

        return root;
    }
};
```

## [将有序数组转换为二叉搜索树](https://leetcode.cn/problems/convert-sorted-array-to-binary-search-tree/)

```cpp
class Solution {
public:
  TreeNode *sortedArrayToBST(vector<int> &nums) {
    return build(nums, 0, nums.size() - 1);
  }

  TreeNode *build(vector<int> &nums, int lo, int hi) {
    if (lo > hi)
      return nullptr;
    int mid = lo + (hi - lo) / 2;
    TreeNode *root = new TreeNode(nums[mid]);
    root->left = build(nums, lo, mid - 1);
    root->right = build(nums, mid + 1, hi);
    return root;
  }
};
```

## [把二叉搜索树转换为累加树](https://leetcode.cn/problems/convert-bst-to-greater-tree/)

```cpp
class Solution {
public:
  TreeNode *convertBST(TreeNode *root) {
    traverse(root);
    return root;
  }

private:
  int sum = 0;
  void traverse(TreeNode *root) {
    if (!root)
      return;
    traverse(root->right);
    sum += root->val;
    root->val = sum;
    traverse(root->left);
  }
};
```

- 涉及到二叉树的构造，无论普通二叉树还是二叉搜索树一定前序，都是先构造中节点。
    
- 求普通二叉树的属性，一般是后序，一般要通过递归函数的返回值做计算。
    
- 求二叉搜索树的属性，一定是中序了，要不白瞎了有序性了。
    

注意在普通二叉树的属性中，我用的是一般为后序，例如单纯求深度就用前序，[二叉树：找所有路径 (opens new window)](https://programmercarl.com/0257.%E4%BA%8C%E5%8F%89%E6%A0%91%E7%9A%84%E6%89%80%E6%9C%89%E8%B7%AF%E5%BE%84.html)也用了前序，这是为了方便让父节点指向子节点。

所以求普通二叉树的属性还是要具体问题具体分析。
