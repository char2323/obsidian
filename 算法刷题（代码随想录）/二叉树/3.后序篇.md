> 前序位置的代码只能从函数参数中获取父节点传来的数据，而后序位置代码不仅可以获取参数数据，还可以获取子树通过函数返回值传递回来的数据。
> 那么换句话说，一旦发现题目和子树有关，那大概率要给函数设置合理的定义和返回值，在后序位置写代码了。

# [寻找重复的子树](https://leetcode.cn/problems/find-duplicate-subtrees/)

> [!question]
> 给你一棵二叉树的根节点 `root` ，返回所有 **重复的子树** 。
> 
> 对于同一类的重复子树，你只需要返回其中任意 **一棵** 的根结点即可。
> 
> 如果两棵树具有 **相同的结构** 和 **相同的结点值** ，则认为二者是 **重复** 的。
> 
> **示例 1：**
> 
> ![](https://assets.leetcode.com/uploads/2020/08/16/e1.jpg)
> 
> **输入：** root = [1,2,3,4,null,2,4,null,null,4]
> **输出：** \[[2,4],[4]\]
> 
> **示例 2：**
> 
> ![](https://assets.leetcode.com/uploads/2020/08/16/e2.jpg)
> 
> **输入：** root = [2,1,1]
> **输出：** \[[1]\]
> 
> **示例 3：**
> 
> **![](https://assets.leetcode.com/uploads/2020/08/16/e33.jpg)**
> 
> **输入：** root = [2,2,2,3,null,3,null]
> **输出：** \[[2,3],[3]\]
> 
> **提示：**
> 
> - 树中的结点数在 `[1, 5000]` 范围内。
> - `-200 <= Node.val <= 200`

```cpp
// 函数签名如下
vector<TreeNode*> findDuplicateSubtrees(TreeNode* root);
```

那么想要知道以自己为根的二叉树是否重复，是否应该被加入到结果列表中，需要知道：
1. 以我为根的这棵二叉树（子树）长什么样？
2. 以其他节点为根的子树都长什么样？
当我知道自己和其他人的样子，就可以对比是否重复了。

## 首先思考：如何才可以知道自己长什么样？

其实这里就可以判断本题需要在后序位置写代码了：因为我需要知道自己的左右子树的样子，再加上自己，就构成二整棵子树的样子。左右子树的样子，正是依赖于在后序位置通过递归函数的返回值传递回来的。

我们可以举一个例子：计算一棵二叉树有多少个节点？很容易写出如下代码：

```cpp
// 定义：输入一棵二叉树，返回二叉树节点的个数
int cout(TreeNode* root){
	if(root==nullptr)
		return 0;
	// 当前节点关心的是两个子树的节点总数分别是多少
	// 因为用子问题的结果可以推导出原问题的结果
	int leftCount = count(root->left);
	int rightCount = count(root->right);
	// 后序位置，左右子树节点树加上自己就是整棵树的节点数
	return leftCount+rightCount+1;
}
```

这就是标准的后序遍历的框架，本质上没有什么区别。

所以现在明确使用后序遍历，那么要如何描述一棵二叉树的样子呢？我们可以知道，二叉树的前中后序遍历的结果可以描述二叉树的结构。

我们以后序遍历结果作为序列化结果，可以通过拼接字符串的方法把二叉树序列化，代码如下：

```cpp
// 定义：输入以 root 为根的二叉树，返回这棵树的序列化字符串
string serialize(TreeNode* root) {
    // 对于空节点，可以用一个特殊字符表示
    if (root == NULL) {
        return "#";
    }
    // 将左右子树序列化成字符串
    string left = serialize(root->left);
    string right = serialize(root->right);
    // 后序遍历代码位置
    // 左右子树加上自己，就是以自己为根的二叉树序列化结果
    string myself = left + "," + right + "," + to_string(root->val);
    return myself;
}
```

我们用非数字的特殊符 `#` 表示空指针，并且用字符 `,` 分隔每个二叉树节点值。这样，我们第一个问题就解决了：对于每个节点，递归函数中的 `myself` 变量就可以描述以该节点为根的二叉树。

接下来考虑第二个问题：
## 其次思考：怎么知道别人长什么样？

很简单，我们可以借助一个外部变量，让每个节点把自己的子树的序列化结果存放进去，这样，对于每个节点，就可以知道有没有其他节点的子树和自己重复了。

初步想法是：使用 HashSet 记录所有子树的序列化结果，可写出如下代码：

```cpp

	// 记录所有子树
	unordered_set<string> subTrees;
	//记录重复子树根节点
	list<TreeNode*>res;

		if(!root)
			return '#';

		string right = serialize(root->right);

		}else{

			subTree.insert(myself);
```

但是这里在出现重复子树的时候，`res` 中必然会有相同的子树重复出现，而题目要求是不希望出现重复。所以我们这里可以把 `HashSet` 升级为 `HashMap`, 额外记录每棵子树出现的次数：

```cpp
class Solution{
public:
	// 记录所有子树以及出现的次数
	unordered_map<string,int> subTrees;
	// 记录重复出现的子树根节点
	vector<TreeNode*> res;
	
	// 主函数
	vector<TreeNode*> findDuplicateSubtrees(TreeNode* root){
		serialize(root);
		return res;
	}
	
	// 辅助函数
	string serialize(TreeNode* root){
		if(!root)
			return "#";
		
		// 先计算左右子树的序列化结果
		string left = serialize(root->left);
		string right = serialize(root->right);
		
		string myself = left + "," + right + "," + to_string(root->val);
		
		int freq = subTrees[myself];
		// 多次重复也只会被加入结果一次
		if(freq==1){
			res.push_back(root);
		}
		
		// 给子树对应的出现次数加一
		subTreeNode[myself] = freq+1;
		return myself;
	}
}
```
