本文考虑对二叉树同时进行「序列化」和「反序列化」。

要说序列化和反序列化，得先从 JSON 数据格式说起。

JSON 的运用非常广泛，比如我们经常将编程语言中的结构体序列化成 JSON 字符串，存入缓存或者通过网络发送给远端服务，消费者接受 JSON 字符串然后进行反序列化，就可以得到原始数据了。

这就是序列化和反序列化的目的，以某种特定格式组织数据，使得数据可以独立于编程语言。

那么假设现在有一棵用 Java 实现的二叉树，我想把它通过某些方式存储下来，然后用 C++ 读取这棵并还原这棵二叉树的结构，怎么办？这就需要对二叉树进行序列化和反序列化了。

# 前言：前中后序和二叉树的唯一性

我们先思考一个问题：

> [!question]
> 什么样的序列化的数据可以反序列化出唯一的一棵二叉树？

我们直接给出结论：

> [!note]
> 当二叉树中节点的值不存在重复时：
> 1. 如果序列化结果中不包含空指针的信息，且只给出了一种遍历顺序，那么无法还原出唯一的一棵二叉树
> 2. 如果序列化结果中不包含空指针的信息，且给出了两种遍历顺序，分两种情况：
> 	1. 如果给出的是前序和中序，或者后序和中序，那么可以还原出唯一的二叉树
> 	2. 如果给出的是前序和后序，那么是无法还原出唯一的二叉树的
> 3. 如果序列化结果中包含空指针的信息，且只给出了一种遍历顺序，也分两种情况：
> 	1. 如果给出的是前序或者后序，那么还原出唯一的二叉树
> 	2. 如果给出的是中序，那么无法还原出唯一的二叉树

关于 3.2 可以这样想：对于这两棵二叉树，他们的中序遍历显然都是 `[#,2,#,1,#]`，是无法区分的。

## 前序遍历解法

前序遍历的示例代码：

```cpp
list<int> res;
void traverse(TreeNode* root) {
    if (root == nullptr) {
        // 暂且用数字 -1 代表空指针 null
        res.push_back(-1);
        return;
    }

    // ****** 前序位置 ********
    res.push_back(root->val);
    // ***********************

    traverse(root->left);
    traverse(root->right);
}
```

调用 `traverse` 函数以后，很容易可以想出 `res` 列表中元素的顺序，比如下图：

![[Pasted image 20260215230348.png]]

那么 `res = [1,2,-1,4,-1,-1,3,-1,-1]`，这就是将二叉树「打平」到了一个列表中，其中 -1 代表 null。

那么将二叉树打平到一个字符串中也是一样的：

```cpp
// 代表分隔符的字符
string SEP = ",";

// 代表 null 空指针的字符
string NULL_CHAR = "#";

// 用于拼接字符串
std::ostringstream os;

// 将二叉树打平为字符串
void traverse(TreeNode* root, std::ostringstream& os) {
    if (root == nullptr) {
        os << NULL_CHAR << SEP;
        return;
    }

    // ***** 前序位置 *****
    os << root->val << SEP;
    // *******************

    traverse(root->left, os);
    traverse(root->right, os);
}
```

这段代码依然是收集前序遍历结果，用 `,` 作为分隔符，用 `#` 表示空指针 null，调用完 `traverse` 函数后，`sb` 中的字符串应该是 `1,2,#,4,#,#,3,#,#,`。

至此，我们已经可以写出序列化函数 `serialize` 的代码了：

```cpp
class Codec {
public:
    string SEP = ",";
    string NULLSYM = "#";

    
    string serialize(TreeNode* root) {
        string sb;
        _serialize(root, sb);
        return sb;
    }

    // 辅助函数，将二叉树存入 StringBuilder
    void _serialize(TreeNode* root, string& sb) {
        if (root == NULL) {
            sb.append(NULLSYM).append(SEP);
            return;
        }

        // ****** 前序位置 ********
        sb.append(to_string(root->val)).append(SEP);
        // ************************

        _serialize(root->left, sb);
        _serialize(root->right, sb);
    }
};
```

现在，思考一下如何写 `deserialize` 函数，将字符串反过来构造二叉树。

首先我们可以把字符串转化成列表：

```cpp
string data = "1,2,#,4,#,#,3,#,#,";
vector<string> nodes;
stringstream ss(data);
string item;
while(getline(ss, item, ',')) {
    nodes.push_back(item);
}
```

这样，`nodes` 列表就是二叉树的前序遍历结果，问题转化为：如何通过二叉树的前序遍历结果还原一棵二叉树？

> [!tip]
> 我们在[[2.构造篇]]中说过，至少要前中后序遍历中的两种配合才可以还原二叉树，那是因为那时的遍历结果中没有记录空指针的信息。这里的 `nodes` 列表包含了空指针的信息，所以只适用 `nodes` 列表就可以还原二叉树。
