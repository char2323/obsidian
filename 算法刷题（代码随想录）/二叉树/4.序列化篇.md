本文考虑对二叉树同时进行「序列化」和「反序列化」。

要说序列化和反序列化，得先从 JSON 数据格式说起。

JSON 的运用非常广泛，比如我们经常将编程语言中的结构体序列化成 JSON 字符串，存入缓存或者通过网络发送给远端服务，消费者接受 JSON 字符串然后进行反序列化，就可以得到原始数据了。

这就是序列化和反序列化的目的，以某种特定格式组织数据，使得数据可以独立于编程语言。

那么假设现在有一棵用 Java 实现的二叉树，我想把它通过某些方式存储下来，然后用 C++ 读取这棵并还原这棵二叉树的结构，怎么办？这就需要对二叉树进行序列化和反序列化了。

# 前言：前中后序和二叉树的唯一性

我们先思考一个问题：

> [!question]
> 什么样的序列化的数据可以反序列化出唯一的一棵二叉树？

我们直接给出结论：

> [!note]
> 当二叉树中节点的值不存在重复时：
> 1. 如果序列化结果中不包含空指针的信息，且只给出了一种遍历顺序，那么无法还原出唯一的一棵二叉树
> 2. 如果序列化结果中不包含空指针的信息，且给出了两种遍历顺序，分两种情况：
> 	1. 如果给出的是前序和中序，或者后序和中序，那么可以还原出唯一的二叉树
> 	2. 如果给出的是前序和后序，那么是无法还原出唯一的二叉树的
> 3. 如果序列化结果中包含空指针的信息，且只给出了一种遍历顺序，也分两种情况：
> 	1. 如果给出的是前序或者后序，那么还原出唯一的二叉树
> 	2. 如果给出的是中序，那么无法还原出唯一的二叉树

**简而言之，分两种情况：**
1. 带有空指针信息的话，中序不能还原，前序和后序可以还原
2. 不带有空指针信息的话，带有中序的两个序列可以还原

关于 3.2 可以这样想：对于这两棵二叉树，他们的中序遍历显然都是 `[#,2,#,1,#]`，是无法区分的。

## 前序遍历解法

前序遍历的示例代码：

```cpp
list<int> res;
void traverse(TreeNode* root) {
    if (root == nullptr) {
        // 暂且用数字 -1 代表空指针 null
        res.push_back(-1);
        return;
    }

    // ****** 前序位置 ********
    res.push_back(root->val);
    // ***********************

    traverse(root->left);
    traverse(root->right);
}
```

调用 `traverse` 函数以后，很容易可以想出 `res` 列表中元素的顺序，比如下图：

![[Pasted image 20260215230348.png]]

那么 `res = [1,2,-1,4,-1,-1,3,-1,-1]`，这就是将二叉树「打平」到了一个列表中，其中 -1 代表 null。

那么将二叉树打平到一个字符串中也是一样的：

```cpp
// 代表分隔符的字符
string SEP = ",";

// 代表 null 空指针的字符
string NULL_CHAR = "#";

// 用于拼接字符串
std::ostringstream os;

// 将二叉树打平为字符串
void traverse(TreeNode* root, std::ostringstream& os) {
    if (root == nullptr) {
        os << NULL_CHAR << SEP;
        return;
    }

    // ***** 前序位置 *****
    os << root->val << SEP;
    // *******************

    traverse(root->left, os);
    traverse(root->right, os);
}
```

这段代码依然是收集前序遍历结果，用 `,` 作为分隔符，用 `#` 表示空指针 null，调用完 `traverse` 函数后，`sb` 中的字符串应该是 `1,2,#,4,#,#,3,#,#,`。

至此，我们已经可以写出序列化函数 `serialize` 的代码了：

```cpp
class Codec {
public:
    string SEP = ",";
    string NULLSYM = "#";
    
    string serialize(TreeNode* root) {
        string sb;
        _serialize(root, sb);
        return sb;
    }

    // 辅助函数，将二叉树存入 StringBuilder
    void _serialize(TreeNode* root, string& sb) {
        if (root == NULL) {
            sb.append(NULLSYM).append(SEP);
            return;
        }

        // ****** 前序位置 ********
        sb.append(to_string(root->val)).append(SEP);
        // ************************

        _serialize(root->left, sb);
        _serialize(root->right, sb);
    }
};
```

现在，思考一下如何写 `deserialize` 函数，将字符串反过来构造二叉树。

首先我们可以把字符串转化成列表：

```cpp
string data = "1,2,#,4,#,#,3,#,#,";
vector<string> nodes;
stringstream ss(data);
string item;
while(getline(ss, item, ',')) {
    nodes.push_back(item);
}
```

这样，`nodes` 列表就是二叉树的前序遍历结果，问题转化为：如何通过二叉树的前序遍历结果还原一棵二叉树？

> [!tip]
> 我们在[[2.构造篇]]中说过，至少要前中后序遍历中的两种配合才可以还原二叉树，那是因为那时的遍历结果中没有记录空指针的信息。这里的 `nodes` 列表包含了空指针的信息，所以只适用 `nodes` 列表就可以还原二叉树。

根据我们的分析，nodes 列表就是一棵打平的二叉树：

![[Pasted image 20260215230348.png]]

那么反序列化过程也是一样的，先确定根节点，然后遵循前序遍历的规则，递归生成左右子树即可：

```cpp
class Codec {
    string SEP = ",";
    string NULLSYM = "#";

    // 主函数，将字符串反序列化为二叉树结构
    TreeNode* deserialize(string data) {
        // 将字符串转化成列表
        list<string> nodes;
        istringstream f(data);
        string s;    
        while (getline(f, s, ',')) {
            nodes.push_back(s);
        }
        return _deserialize(nodes);
    }

    // 辅助函数，通过 nodes 列表构造二叉树
    TreeNode* _deserialize(list<string>& nodes) {
        if (nodes.empty()) return nullptr;

        // ****** 前序位置 ********
        // 列表最左侧就是根节点
        string first = nodes.front();
        nodes.pop_front();
        if (first == NULLSYM) return nullptr;
        TreeNode* root = new TreeNode(stoi(first));
        // *********************

        root->left = _deserialize(nodes);
        root->right = _deserialize(nodes);

        return root;
    }
};
```

所以这里我们利用树的递归的性质，`nodes` 列表的第一个元素就是一棵树的根节点，所以只需要将列表的第一个元素取出作为根节点，余下的交给递归函数处理即可。

## 后序遍历解法

二叉树后序遍历的框架：

```cpp
void traverse(TreeNode* root){
	if(root==nullptr)
		return;
	traverse(root->left);
	traverse(root->right);
	
	// 后序位置的操作代码
}
```

有了前序遍历以后，后序遍历就相对简单了。`serialize` 序列化方法非常容易实现，只需要稍作修改前文的 `_serialize` 辅助方法即可：

```cpp
// 辅助函数，将二叉树存入 StringBuilder
void _serialize(TreeNode* root, StringBuilder& sb) {
    if (root == nullptr) {
        sb.append(NULL).append(SEP);
        return;
    }
    
    _serialize(root->left, sb);
    _serialize(root->right, sb);

    // ****** 后序位置 ********
    sb.append(root->val).append(SEP);
    // ***********************
}
```

我们把对 `StringBuilder` 的拼接操作放到了后序遍历的位置，后序遍历导致结果的顺序发生变化：

![[Pasted image 20260309150234.png]]

关键是如何实现后序遍历的 `deserialize` 方法呢？

在上图中，我们能看到 `root` 节点位于列表的最后一个位置，所以我们要从后往前提取列表元素，先用最后一个元素构造 `root`, 然后递归调用生成左右子树。但有一个地方需要注意：一定要先构造 `root.right` 子树，然后再构造 `root.left` 子树。

```cpp
class Codec {
public:
    string SEP = ",";
    string NULL_STR = "#";

    // 主函数，将二叉树序列化为字符串
    string serialize(TreeNode* root) {
        string sb;
        _serialize(root, sb);
        return sb;
    }

    void _serialize(TreeNode* root, string& sb) {
        if (root == nullptr) {
            sb += NULL_STR + SEP;
            return;
        }
        
        _serialize(root->left, sb);
        _serialize(root->right, sb);

        // ****** 后序位置 ********
        sb += to_string(root->val) + SEP;
        // ***********************
    }

    // 主函数，将字符串反序列化为二叉树结构
    TreeNode* deserialize(string data) {
        list<string> nodes;
        stringstream ss(data);
        string buf;
        while (getline(ss, buf, ',')) {
            nodes.push_back(buf);
        }
        return _deserialize(nodes);
    }

    // 辅助函数，通过 nodes 列表构造二叉树
    TreeNode* _deserialize(list<string>& nodes) {
        if (nodes.empty()) return nullptr;
        // 从后往前取出元素
        string last = nodes.back();
        nodes.pop_back();
        if (last == NULL_STR) return nullptr;
        TreeNode* root = new TreeNode(stoi(last));
        // 先构造右子树，后构造左子树
        root->right = _deserialize(nodes);
        root->left = _deserialize(nodes);
        
        return root;
    }
};
```

以上就是后序遍历的序列化和反序列化方法了。

## 中序遍历解法

很可惜，中序遍历实现不了反序列方法 `deserialize`。序列化方法 `serialize` 依然很容易。只需要把辅助函数 `_seialize` 中拼接字符串的操作放到中序遍历的位置即可：

```cpp
// 辅助函数，将二叉树存入 StringBuilder
void serialize(TreeNode* root, string &sb) {
    if (root == nullptr) {
        sb.append("NULL").append(SEP);
        return;
    }

    serialize(root->left, sb);
    // ****** 中序位置 ******
    sb.append(to_string(root->val)).append(SEP);
    // *********************
    serialize(root->right, sb);
}
```

我们知道要想实现反序列化方法，首先要构造根节点 `root`, 在前序遍历得到的 `nodes` 列表中，第一个元素就是 `root` 节点的值；后序遍历得到的 `nodes` 列表中，最后一个元素就是 `root` 节点的值。然而在中序遍历中，`root` 的值被夹在两棵子树之间，不知道确切的索引位置，所以无法找到根节点 `root`, 也就无法反序列化。

## 层序遍历解法

二叉树层序遍历的代码框架：

```cpp
void traverse(TreeNode* root) {
    if (root == nullptr) return;
    
    // 初始化队列，将 root 加入队列
    queue<TreeNode*> q;
    q.push(root);
    
    while (!q.empty()) {
        int sz = q.size();
        for (int i = 0; i < sz; i++) {
            // 层级遍历代码位置
            TreeNode* cur = q.front();
            q.pop();
            cout << cur -> val << endl;
            // ***************
            if (cur -> left != nullptr) {
                q.push(cur -> left);
            }
            if (cur -> right != nullptr) {
                q.push(cur -> right);
            }
        }
    }
}
```

这很基础，从上到下，从左到右打印每一层二叉树节点的值。但可以看到，队列 `q` 中不会存储 `null` 指针。然而我们在反序列化过程中是需要记录空指针的，所以可以稍作修改：

```cpp
#include<queue>

void traverse(TreeNode* root) {
    if (root == nullptr) return;
    // 初始化队列，将 root 加入队列
    queue<TreeNode*> q;
    q.push(root);

    while (!q.empty()) {
        int sz = q.size();
        for (int i = 0; i < sz; i++) {
           TreeNode* cur = q.front();
           q.pop();
           // 层级遍历代码位置
           if (cur == nullptr) continue;
           cout << cur->val << endl;

           q.push(cur->left);
           q.push(cur->right);
        }
        
    }
}
```

这样也是可以完成层序遍历的，不过我们就是把对空指针的检验从“将元素加入队列”的时候改为了“从队列中取出元素”的时候。这样就可以得到序列化方法：

```cpp
class Codec {
private:
    const string SEP = ",";
    const string NULL = "#";

public:
    // 将二叉树序列化为字符串
    string serialize(TreeNode* root) {
        if (root == nullptr) return "";
        string sb;
        // 初始化队列，将 root 加入队列
        queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            int sz = q.size();
            for (int i = 0; i < sz; ++i) {
                TreeNode* cur = q.front();
                q.pop();
                // 层级遍历代码位置
                if (cur == nullptr) {
                    sb += NULL + SEP;
                    continue;
                }
                sb += to_string(cur->val) + SEP;
                // ***************
                q.push(cur->left);
                q.push(cur->right);
            }
        }
        return sb;
    }
};
```

可以看到每一个非空节点对应两个字节点，那么反序列化的思路也是利用队列进行层级遍历：

```cpp
class Codec {
    string SEP = ",";
    string NULLSYM = "#";
  
    // 将字符串反序列化为二叉树结构
    public:
    TreeNode* deserialize(string data) {
        if (data.empty()) return nullptr;
        stringstream ss(data);
        string item;
        getline(ss, item, ',');
        // 第一个元素就是 root 的值
        TreeNode* root = new TreeNode(stoi(item));
        // 队列 q 记录父节点，将 root 加入队列
        queue<TreeNode*> q;
        q.push(root);

        while (!q.empty()) {
            int sz = q.size();
            for (int i = 0; i < sz; i++) {
                TreeNode* parent = q.front();
                q.pop();
                // 为父节点构造左侧子节点
                if(!getline(ss, item, ',') || item == NULLSYM) {  
                    parent->left = nullptr;  
                } else {
                    parent->left = new TreeNode(stoi(item));
                    q.push(parent->left);
                }
                // 为父节点构造右侧子节点
                if(!getline(ss, item, ',') || item == NULLSYM) {  
                    parent->right = nullptr;  
                } else {
                    parent->right = new TreeNode(stoi(item));
                    q.push(parent->right);
                }
            }
        }
        return root;
    }
};
```
