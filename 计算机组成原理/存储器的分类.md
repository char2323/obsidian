# 存储器全维度分类导图

## 按在计算机中的作用 (层次结构)

### 主存储器 (Main Memory)

- 特性: CPU 能直接访问，速度较快，容量适中。
    
- 考点: 包含 RAM 和 ROM；考查地址映射与引脚数计算。
    

### 辅助存储器 (Auxiliary Memory)

- 特性: 存在于主机之外，容量大，速度慢，非易失。
    
- 考点: 磁盘、光盘、磁带；属于 I/O 设备，不直接与 CPU 交换数据。
    

### 高速缓冲存储器 (Cache)

- 特性: 介于 CPU 和主存之间，解决速度不匹配问题。
    
- 考点: 使用 SRAM；考查命中率、局部性原理及替换算法。
    

### 寄存器 (Register)

- 特性: 存在于 CPU 内部，速度最快，容量极小。
    
- 考点: 指令寄存器 (IR)、程序计数器 (PC) 等。
    

## 按存取方式

### 随机存取 (Random Access - RAM)

- 特性: 存取时间与物理位置无关 ($O(1)$)。
    
- 考点: 包含 SRAM (Cache) 和 DRAM (主存)。
    

### 只读存取 (Read-Only - ROM)

- 特性: 结构同 RAM，但通常只读（现代 ROM 可电擦写）。
    
- 考点: Flash (闪存) 虽然属于 ROM 家族，但支持块擦除；非易失性。
    

### 串行存取 (Sequential/Direct Access)

- **顺序存取 (SAM)**
    
    - 特性: 必须按物理位置先后寻找（如磁带）。
        
    - 考点: 存取时间依赖于物理位置，速度最慢。
        
- **直接存取 (DAM)**
    
    - 特性: 先跳跃寻道，再顺序访问（如磁盘）。
        
    - 考点: 存取时间 = 寻道时间 + 延迟时间 + 传输时间。
        

## 按存储介质

### 半导体存储器

- 特性: 体积小、速度快。
    
- 考点: 分为双极型 (快, 功耗大) 和 MOS 型 (集成度高, 考研主角)。
    

### 磁表面存储器

- 特性: 磁层存储，非易失，容量大。
    
- 考点: 磁盘、磁带；需考虑磁头定位。
    

### 光存储器

- 特性: 激光读写，记录密度高。
    
- 考点: CD-ROM、DVD。
    

## 按信息还原性 (持久性)

### 易失性存储器 (Volatile)

- 特性: 断电后信息立刻消失。
    
- 考点: RAM (SRAM, DRAM) 均为此类。
    

### 非易失性存储器 (Non-volatile)

- 特性: 断电后信息依然保留。
    
- 考点: ROM、磁表面存储器。
    

## 对比 (关键考点)

### SRAM vs DRAM

- SRAM: 触发器存储 (6 T)；不需刷新；快；贵；Cache。
    
- DRAM: 电容存储 (1 T)；**必须刷新**；破坏性读出；慢；廉价；主存。
    

### DRAM 刷新方式

- 集中刷新: 有“死区”。
    
- 分散刷新: 无“死区”，但存取周期长。
    
- **异步刷新 (最常考)**: 每行在规定时间内刷一次；死区分布开，效率高。
    

### 存储器地址转换

- 考点: 掌握 MAR 位数与寻道空间的关系；区分字节寻址与字寻址。
